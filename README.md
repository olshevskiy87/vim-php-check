# vim-php-check

Tiny PHP code checking plugin for Vim.

## Installation

vim-plug:

```vim
Plug 'olshevskiy87/vim-php-check'
```

## Usage

- `<leader>ps`, `:call PhpStanCheck([<level>])` - check file with phpstan
- `<leader>pm`, `:call PhpMdCheck()` - check file with phpmd
